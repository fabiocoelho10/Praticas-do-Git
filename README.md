# Praticas-do-Git
## Como executar o arquivo .sh
1. Clone o repertório:
   git clone https://github.com/fabiocoelho10

2. Entre na pasta:
   cd Praticas-do-Git

3. Dê permissão de execução:
   chmod +x executar.sh

4. Execute:
   ./executar.sh

   ## Sobre o código Python

   O arquivo 'my_first_calculator.py' é uma calculadora simples feita em python.
   Ela permite realizar operações básicas como soma, subtração, multiplicação e divisão.

   ### Requisitos
   - Python 3.x instalado

   ### Como executar diretamente
   python3 my_first_calculator.py
